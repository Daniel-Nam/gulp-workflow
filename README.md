# Basic Gulp-Workflow for beginners 🔥🔥🔥

## ⚙️ Technology Using

-   `gulp`
-   `gulp-babel`
-   `autoprefixer`
-   `gulp-postcss`
-   `cssnano`
-   `gulp-pug`
-   `gulp-sass`
-   `sass`
-   `browser-sync`

### 🤖 Require

-   ✅ _Installed Nodejs_
-   ✅ _Download all Dependencies before using_
-   💀 **Don't edit anything in the public folder, you can change the images folder if you like :v**

### 📃 Usage

```
npm install - download Dependencies
gulp - start app
```
